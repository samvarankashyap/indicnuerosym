# Synthetic Data

LLM-generated Dwipada poems and supporting datasets used for augmentation and curriculum learning.

## Files

| File | Description |
|---|---|
| `combined_dwipada_poems.json` | Combined synthetic poems from all contributors (4.8MB) |
| `kannada_ragale.json` | Kannada ragale dataset for cross-script experiments (280KB) |
| `samvaran_1000_synthetic_dwipada.txt` | 1000 synthetic Dwipada poems (2.1MB) |
| `nlp_mahesh.txt` | Contributor synthetic dataset (Mahesh) |
| `nlp_pradeep.txt` | Contributor synthetic dataset (Pradeep) |
| `nlp_favi_kannadadataset.txt` | Contributor Kannada dataset (Favi) |
