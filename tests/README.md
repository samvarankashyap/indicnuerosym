# Tests

Unit and integration tests for the dwipada package.

## Files

| File | Description |
|---|---|
| `test_analyzer.py` | Prosody analysis tests — gana partitioning, yati/prasa validation |
| `test_imports.py` | Package import smoke tests |
| `conftest.py` | pytest configuration and shared fixtures |

## Running

```bash
pytest tests/
```
