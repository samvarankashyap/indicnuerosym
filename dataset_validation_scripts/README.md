# Dataset Validation Scripts

Multi-level quality validation for the Dwipada augmented dataset.

## Validation Levels

1. **Chandass Scanner** — Structural integrity (gana/yati/prasa checks)
2. **Sanity Checks** — Coarse alignment of Telugu/English meanings
3. **Embedding-based Checks** — Semantic similarity validation
4. **Format Validation** — JSON schema compliance

## Files

| File | Description |
|---|---|
| `validation_report.txt` | Comprehensive 4-level validation report with pass/fail statistics |
